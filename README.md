# 🎵 No One Noticed - Interactive Visualizer

A minimalist, Apple Music-inspired web experience built with:
* **HTML5 / CSS3** (Glassmorphism & Keyframe Animations)
* **Vanilla JavaScript** (DOM Manipulation & Audio Synchronization)

### Features
* **Dynamic Lyrics:** Synchronized scrolling based on audio timestamps.
* **Interactive UI:** Click/Tap anywhere to generate floating `heart.jpg` particles.
* **Responsive Design:** Optimized for both mobile and desktop views.
